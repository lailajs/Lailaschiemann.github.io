Among Us
Simulated the popular game "Among Us" with an Impostor mode and Ghost mode. Used Minimum Spanning Trees, Prims Algorithm, Branch and Bound Algorithm, and TSP Heuristics to find the best possible path between rooms given different player modes. Wrote test cases from scatch to debug code.
SillyQL
Wrote program to emulate a relational database using a standard query language as the interface. Designed and planned multiple functions using differing abstract data types such as unordered maps as the back bone of hash functions, functors, maps as the backbone of binary search trees, vectors and original class types to efficiently evaluate different representations of the same data, and create a program that utilized efficent runn time and memory storage.
Mine all Mine
Used several kinds of priority queues, inheritance and dynamic polymorphism to escape from a mine using dynamite sticks. Needing to make the most of each dynamite stick, I used these data structures to blast piles of rubble that would take the fewest sticks of dynamite to destroy. Wrote test cases from scatch to debug code.
